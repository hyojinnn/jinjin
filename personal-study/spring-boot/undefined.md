# 환경설정



**스프링 부트의 특징**

* 일반적인 응용 프로그램을 단독으로 실행하는 수준으로 스프링 애플리케이션을 구현할 수 있습니다.
* 프로젝트 환경을 구축할 때 필요한 톰캣, Jetty, UnderFlow 같은 외적인 툴이 내장되어 있어 따로 설치할 필요가 없습니다.
* XML 기반 설정이나 코드 없이 환경 설정을 자동화할 수 있습니다.
* 의존성 관리를 쉽게 자동으로 할 수 있습니다.
